DeviceOwnerAuthentication

public enum DeviceOwnerAuthentication : CaseIterable, Sendable

Indicates whether the device is set up with passcode or biometrics.

  • biometricNotSet

    This is returned when CardHolderVerificationMethod is configured with biometricOnly and biometric is not enrolled on the device.

    Declaration

    case biometricNotSet
  • passcodeNotSet

    This is returned when CardHolderVerificationMethod is configured with userPresence and passcode is not set on the device.

    Declaration

  • supported

    This is returned when the device is properly set up with passcode and biometric, or only passcode, according to the selected CardHolderVerificationMethod.
