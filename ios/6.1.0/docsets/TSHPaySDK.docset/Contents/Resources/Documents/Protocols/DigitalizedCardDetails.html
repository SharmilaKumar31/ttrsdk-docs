@protocol DigitalizedCardDetails <NSObject>

protocol DigitalizedCardDetails : NSObjectProtocol

DigitalizedCardDetails @discussion Protocol that will be used to retrieve details about the card that has been provisioned. It is through this protocol where the user of the SDK is able to retrieve the card information in the card profile.
