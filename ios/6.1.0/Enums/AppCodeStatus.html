Swift

enum AppCodeStatus : UInt

Enumeration indicating the app code status.

Since

Since 6.0.0
  • AppCodeStatusNotSupported

    There is no card exists or the card profile is not supporting flexible CD CVM. or the access control is not an application password.

    Declaration

    Objective-C

    AppCodeStatusNotSupported = 1

    Swift

  • AppCodeStatusNotExisting

    The app code is not set. This could also happen if when performing app code related operations but the app code is not set.

    Declaration

    Objective-C

    Swift

    This could happen if the encryption algorithm (kSecKeyAlgorithmRSAEncryptionOAEPSHA256) is not supported on the device.
