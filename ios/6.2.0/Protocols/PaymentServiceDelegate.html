  • -onPaymentServiceActivate:chVerification:cvmTimeout:

    This method is called when the card has been successfully loaded. Upon receipt of this call-back, the Library User needs to verify the cvm. The Library User needs to

    • Acquire the currently activated PaymentService from PaymentBusinessService. On the interface of the PaymentService, the Library User can get the appropriate verifier object for verification
    • : Also, after the End User has input the credentials, the credentials shall time out (wiped away for security reason) after a period (the duration is in the parameter cvmResetTimeout). The Library User may want to show a count down clock and listen to this event by registering through cvmResetTimeoutDelegate

    Declaration

                      cvmTimeout:(uint)cvmTimeout;

    Parameters

    verifier

    cvmTimeout

    Timeout duration after the CH has been verified in seconds.
